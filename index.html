🔧 Konfigurimi i Google Drive

Për të aktivizuar sinkronizimin, duhet të vendosni Google OAuth Client ID tuajin. Klik këtu për udhëzimet.

📘 Si të Krijoni Client ID

1. Shko te console.cloud.google.com dhe krijo një projekt të ri (p.sh. "Regjistri").

2. Aktivizo Google Drive API: meny → "APIs & Services" → "Library" → kërko "Google Drive API" → "Enable".

3. Shko te "OAuth consent screen" → zgjedh "External" → plotëso emrin dhe email-in → ruaj.

4. Shko te "Credentials" → "Create Credentials" → "OAuth client ID" → zgjedh "Web application".

5. Tek "Authorized JavaScript origins" shto

6. Kopjo Client ID dhe ngjite në kutinë e konfigurimit.

7. Te "OAuth consent screen" → "Test users" → shto email-in tënd që të hyrë.
